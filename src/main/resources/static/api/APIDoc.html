Rooms

Знайти квартири по параметрам.

GET /api/rooms

Пошук квартир по параметрам (Площа квартири, кількість спальних кімнат, ціна оренди).

Query parameters
  • area number

    Площа квартири.

  • bedroomCount integer

    Кількість спальних кімнат.

  • price integer

    Ціна оренди.

  • page integer

    Сторінка.

    Default value is 0.

  • size integer

    Розмір сторінки.

    Default value is 3.

Responses
  • • 200 array[object]

    Успішний пошук квартири.

    • id integer(int64)
    • configuration object
      • id integer(int64)
      • area number
      • bedroomCount integer
      • price number
    • description object
      • id integer(int64)
      • description string
      • address string
      • creationDate string(date-time)
    • user object
      • email string
      • fullName string

Додати нову квартиру до списку.

POST /api/rooms

Додання квартири до списку існуючих.

Query parameters
  • configuration.area Required / number

    Площа квартири.

  • configuration.bedroomCount Required / integer

    Кількість спальних кімнат.

  • configuration.price Required / integer

    Ціна оренди.

  • description.description Required / string

    Опис квартири.

  • description.address Required / string

    Адреса.

Responses
  • • 200 boolean

    Успішне додання квартири.

  • • 400

    Передано невалідні значення.

  • • 401

    Немає Авторизації.

POST /api/rooms 
curl \
 -X POST http://localhost:8080/api/rooms?configuration.area=42.0&configuration.bedroomCount=42&configuration.price=42&description.description=string&description.address=string
Response example (200) 
true

Отримання квартири по id

GET /api/rooms/{id}

Отримання квартири по id з повним описом.

Path parameters
  • id Required / number

    id квартири.

Responses
  • • 200 object

    Успішний вивід квартири.

    • id integer(int64)
    • configuration object
      • id integer(int64)
      • area number
      • bedroomCount integer
      • price number
    • description object
      • id integer(int64)
      • description string
      • address string
      • creationDate string(date-time)
    • user object
      • email string
      • fullName string
  • • 404

    Не знайдено кімнати за заданим id.

Оновлення інформації про квартиру.

PUT /api/rooms/{id}

Перезапис інформації про існуючу квартиру новими даними.

Path parameters
  • id Required / number

    id квартири.

Query parameters
  • configuration.area Required / number

    Площа квартири.

  • configuration.bedroomCount Required / integer

    Кількість спальних кімнат.

  • configuration.price Required / integer

    Ціна оренди.

  • description.description Required / string

    Опис квартири.

  • description.address Required / string

    Адреса.

Responses
  • • 200 boolean

    Успішне оновлення квартири.

  • • 400

    Передано невалідні значення.

  • • 401

    Немає Авторизації.

  • • 404

    Не знайдено кімнати за заданим id.

PUT /api/rooms/{id} 
curl \
 -X PUT http://localhost:8080/api/rooms/{id}?configuration.area=42.0&configuration.bedroomCount=42&configuration.price=42&description.description=string&description.address=string
Response example (200) 
true
